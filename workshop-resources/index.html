Cowork: Connectors

Excalidraw, Canva & Plugins

✎
Try Excalidraw Connector
Cowork → Customize → Connectors → Add Excalidraw → Start a new task.
Copy-paste prompt
Create an Excalidraw diagram of my business workflow. My business is [describe in 1 sentence]. Show the main steps from customer discovery to delivery. Use friendly icons and colors.
🎨
Try Canva Connector
Cowork → Customize → Connectors → Add Canva → Log in → New task.
Copy-paste prompt
Create a professional Instagram post for my business [your business name]. Use Canva to design it. The post should announce [a promotion, event, or new product]. Use modern, clean design with bold text.
🔌
Try a Plugin (Web Search)
Cowork → Customize → Plugins → Toggle on Web Search → New task.
Copy-paste prompt
Search the web for the top 3 trends in [your industry] in 2026. Summarize them in a table with what it means for my business.

Chrome Extension
No file needed. Just paste the prompt. Claude builds a working Chrome extension you can install in 2 minutes.
Copy-paste prompt
Build me a Chrome extension that does the following: - Adds a button to save any webpage as a clean, readable summary - When clicked, it extracts the page title, URL, and main text content - Saves it to a list I can view in the extension popup - Shows a "Saved!" notification - Include clear instructions for how to install it in Chrome developer mode Create all the files: manifest.json, popup.html, popup.js, content.js, and an icon.
