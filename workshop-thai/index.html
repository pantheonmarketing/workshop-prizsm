7 สิ่งที่คุณทำได้
ใน Cowork ตอนนี้เลย

จะสาธิตทีละอัน แล้วคุณลอง

Use Case 1

🗃 Digital Janitor (คนทำความสะอาดไฟล์)

Claude อ่านทุกไฟล์ในโฟลเดอร์รก เข้าใจว่าแต่ละไฟล์คืออะไร เปลี่ยนชื่อ และจัดเข้าโฟลเดอร์

⬇ ดาวน์โหลดไฟล์ตัวอย่าง
Prompt: Look at this file list. Read each description to understand what each file actually is. Rename every file with a clear, descriptive name. Then organize them into logical folders. Show me the before and after.

Use Case 2

📊 Data → Charts (ข้อมูลเป็นกราฟ)

อัพโหลด Spreadsheet หรือ CSV Claude เขียน Python รันให้ ได้กราฟสวยๆ + รายงานเต็มรูปแบบ

⬇ ดาวน์โหลด Sample CSV
Prompt: Analyze this sales data CSV. Create bar charts for revenue by month, a line chart for trends, and a pie chart for categories. Write a 1-page executive summary with key insights. Save everything as files I can download.

Use Case 3

📄 Bulk Invoice Processing (ใบแจ้งหนี้)

โยน PDF ใบแจ้งหนี้ 50+ ใบ Claude อ่านทุกใบ ดึงข้อมูล สร้างตาราง Master

⬇ ดาวน์โหลดใบแจ้งหนี้ตัวอย่าง
Prompt: Process all the invoices in this file. For each one, extract: invoice number, vendor name, date, due date, and total amount (THB). Create a master CSV spreadsheet. Sort by date.
📊

ลองเลย

ดาวน์โหลด CSV อัพโหลดเข้า Cowork ได้กราฟทันที

⬇ ไปหน้าดาวน์โหลด
วาง Prompt นี้: Analyze this sales data. Create charts for revenue by month and by category. Write a short summary with the key insights. Save as downloadable files.

Use Case 4

🎯 Customer List → Visual CRM

โยนรายชื่อลูกค้า Claude สร้าง Kanban board ลากวางได้ มีสถานะ: New, Contacted, Proposal, Won, Lost

⬇ ดาวน์โหลดรายชื่อลูกค้า
Prompt: Build me a visual CRM pipeline as a single HTML file. Use this customer list. Create a Kanban board with columns: New Lead, Contacted, Proposal Sent, Won, Lost. Make each card draggable. Show customer name, deal value, and status.
🎯

ลองเลย

ดาวน์โหลดรายชื่อลูกค้า อัพโหลด ได้ CRM ทันที

⬇ ไปหน้าดาวน์โหลด
วาง Prompt นี้: Build me a visual CRM pipeline from this customer data. Create a Kanban board with columns: New Lead, Contacted, Proposal, Won, Lost. Make it drag-and-drop. Save as an HTML file.

Use Case 5

📧 Newsletter Generator

บอก Claude เกี่ยวกับธุรกิจ ได้ Newsletter HTML สวยๆ พร้อม Copy ไปใส่ Gmail หรือ Mailchimp

⬇ ดาวน์โหลด Business Brief
Prompt: Read this business brief. Create a professional HTML newsletter email with: a welcome header, main announcement, special offer section, CTA button, and footer. Use brand colors from the brief.

Use Case 6

🌐 Build a Website (สร้างเว็บไซต์)

อธิบายธุรกิจ Claude สร้างเว็บไซต์ครบ: Hero, About, Services, Contact Form ในไฟล์ HTML เดียว

Prompt: Build me a complete one-page website for my [ธุรกิจ] called [ชื่อ]. Include hero, about, 3 services, testimonials, and contact form. Use [สีที่ต้องการ]. Save as HTML.

Use Case 7

🧹 Chrome Extension

อธิบายเครื่องมือ Browser ที่อยากมี Claude สร้าง Chrome Extension ทั้งตัว โหลดใน Developer Mode ใช้ได้ทันที

Prompt: Build me a Chrome extension that saves any webpage as a clean PDF with one click. Include an icon, a popup that shows "Saved!" confirmation, and instructions for how to install it.
🌐

ลองเลย
